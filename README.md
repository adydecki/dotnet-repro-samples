# .NET MAUI Repro Samples

This repository contains minimal reproduction projects for various issues reported to the [dotnet/maui](https://github.com/dotnet/maui) repository. Each sample is designed to be isolated and focused on a specific bug.

## 📂 Project Structure

Each reproduction is contained within its own folder, named according to the GitHub issue number and a brief description:

- `/Issue-XXXXX-Short-Description`: Focused project for issue #XXXXX.
- `/Issue-YYYYY-Short-Description`: Focused project for issue #YYYYY.
